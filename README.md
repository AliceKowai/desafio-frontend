# 🎯 Desafio Frontend Junior - Card de Perfil Interativo

Este projeto foi desenvolvido como parte do processo seletivo da **Frontend Fusion**. O objetivo foi criar um componente de card de perfil moderno, funcional e fiel ao design proposto no Figma.

## 🚀 Tecnologias Utilizadas

O projeto foi construído utilizando as tecnologias mais atuais do ecossistema React:

* **Next.js 14** (App Router)
* **React** (Hooks para gerenciamento de estado)
* **Tailwind CSS** (Estilização utilitária e responsiva)
* **TypeScript** (Tipagem estática para maior segurança do código)
* **Lucide React / SVGs Customizados** (Ícones fiéis ao design)

## ✨ Funcionalidades e Diferenciais

O desafio foi implementado seguindo o conceito de **Pixel Perfect**, garantindo que cada medida e cor corresponda ao design de referência:

* ✅ **Componente Reutilizável:** `ProfileCard` configurado para receber dados via props.
* ✅ **Estado de Seguidores:** Lógica funcional que incrementa/decrementa o contador ao clicar em "Seguir".
* ✅ **Interatividade Avançada:** * Botão "Seguir" com troca dinâmica de estilos (Degradê para Verde Sólido).
    * Aplicação de sombras projetadas específicas (`shadow-[#10B981]/30`).
    * Efeitos de `hover` e `active` para melhor experiência do usuário (UX).
* ✅ **Fidelidade Visual:** Avatar com borda de `4px` (#A241C2) e dimensões exatas de `146px`.
* ✅ **Deploy:** Aplicação publicada e funcional na Vercel.

## 🛠️ Como executar o projeto localmente

1.  **Clone o repositório:**
    ```bash
    git clone [https://github.com/seu-usuario/seu-repositorio.git](https://github.com/seu-usuario/seu-repositorio.git)
    ```

2.  **Instale as dependências:**
    ```bash
    npm install
    ```

3.  **Inicie o servidor de desenvolvimento:**
    ```bash
    npm run dev
    ```

4.  **Acesse no navegador:**
    [http://localhost:3000](http://localhost:3000)
